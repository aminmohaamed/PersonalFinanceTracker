## ğŸ“± Main Features

### Dashboard
- Total balance overview

### Add Transaction
1. Click "Add Transaction" in sidebar
2. Select Income or Expense
3. Choose category
4. Enter description and amount
5. Select date
6. Click "Save Transaction"

### Manage Budgets
1. Click "Budgets" in sidebar
2. Click "Create Budget"
3. Select category (expense categories only)
4. Set monthly limit
5. Choose month and year
6. Click "Create Budget"

### View History
1. Click "Transactions" in sidebar
2. Use filters to find specific transactions:
   - Date range
   - Category
   - Type (Income/Expense)
3. Edit or delete transactions as needed
